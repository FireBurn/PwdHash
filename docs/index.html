PwdHash - Modern Password Generator

This page generates deterministic, site-specific passwords using the secure PBKDF2 algorithm.

1. Enter Site URL or Domain

e.g., "gitlab.freedesktop.org"

Domain Being Hashed

2. Enter Master Password

This is the only password you need to remember.

3. Your Hashed Password

This tool implements a modern password hashing scheme using PBKDF2 with SHA-256 via the Web Crypto API. All calculations are done locally in your browser. Nothing is ever sent to a server.
